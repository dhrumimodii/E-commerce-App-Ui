##Flutter Product Grid App

A simple Flutter app that fetches product data from an API and displays it in a responsive grid layout.

## Features

* Fetch data from REST API
* Display products in grid view
* Show product image, title, and price
* Loading indicator while fetching data

##  Tech Stack

* Flutter
* Dart
* HTTP package
* REST API (Fake Store API)

##  Run the App

```bash
git clone https://github.com/your-username/flutter-product-app.git
cd flutter-product-app
flutter pub get
flutter run
```
